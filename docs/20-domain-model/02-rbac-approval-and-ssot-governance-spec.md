# RBAC, Approval and SSoT Governance Spec

## 역할
- org_owner
- workspace_admin
- editor_reviewer
- author
- viewer

## visibility
- private
- workspace
- organization
- ssot_pinned

## Template 상태
draft → published → ssot_candidate → certified_ssot → deprecated → archived

## 핵심 원칙
- 편집 권한과 인증 권한은 다르다.
- SSoT는 거버넌스 상태다.
- 고위험 상태 전이는 audit 가능해야 한다.
- 실행 권한과 자산 관리 권한은 분리한다.
