# TASKFLOW Document OS Docs

이 폴더는 **TASKFLOW 문서 OS 생태계 시스템**의 제품, 도메인, Wizard, 런타임, 스튜디오 앱, 아키텍처, 보안/운영, 실행 백로그에 관한 **정본(reference source of truth)** 문서를 담는 공간이다.

TASKFLOW는 단순 프롬프트 편집기가 아니다. 이 시스템은 TF8 기반의 공통 문서 운영체제(Core OS) 위에, 보도자료 스튜디오, 회의록 스튜디오, 공지 이메일 스튜디오 같은 장르별 실행 앱(Studio Apps)을 얹는 구조를 가진다.
