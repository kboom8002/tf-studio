# Vision, Scope and Non-goals

## Vision
자연어 프롬프트를 개인의 감각이 아니라 조직의 시스템으로 전환한다.

## Scope
- TF8 기반 Core OS
- 6단계 Wizard
- Prompt Assembly
- Sanitization
- Main Run / Review Run
- Prompt Hub
- SSoT 인증
- Press Release / Meeting Minutes / Announcement Email Studio

## Non-goals
- 실제 배포/발송 시스템
- 언론 CRM / 메일 CRM
- 실시간 공동 편집
- 고급 block drag-and-drop composer 완성형
