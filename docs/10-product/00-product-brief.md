# Product Brief 1.0

**TASKFLOW 문서 OS는 TF8 기반의 공통 프롬프트 운영체제를 중심으로, 보도자료·회의록·공지 이메일 등 업무 문서 장르별 실행 스튜디오를 제공하는 Enterprise Prompt Engineering SaaS이다.**

## 핵심 문제
- 문서 작성이 개인 역량에 과도하게 의존
- 생성형 AI를 써도 품질과 보안이 통제되지 않음
- 좋은 생산 방식이 조직 자산이 되지 않음

## 핵심 구조
### Core OS
- TF8 스키마
- 상수/변수 분리
- 6단계 Wizard
- K 블록 비식별화
- 메인 생성 + Red-Team Reviewer
- Prompt Hub / SSoT
- RBAC / 감사 로그 / 측정 계층

### Studio Apps
- Press Release Studio
- Meeting Minutes Studio
- Announcement Email Studio

## 핵심 가치
1. 입력을 통제한다
2. 출력을 검수한다
3. 좋은 생산 방식을 자산화한다
