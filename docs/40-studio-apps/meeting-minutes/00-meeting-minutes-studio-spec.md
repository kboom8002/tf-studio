# Meeting Minutes Studio Spec

## 목적
회의 메모/녹취를 구조화된 회의록과 실행 항목으로 정리한다.

## TF8 매핑
- T: 회의록 및 결정사항 정리
- A: 참석자 / 공유 대상 / 실행자
- S: 회의 목적, 프로젝트 맥락
- K: 메모, 녹취, 안건, 참석자 정보
- F: 개요 → 안건 → 주요 논의 → 결정사항 → 액션아이템 → 책임자/기한 → 미해결 이슈
- L: 중립적 업무 기록체
- O: 구조화 회의록
- W: 임의 추가 금지, 책임자/기한 누락 경계

## 출력 계약
- 회의 개요
- 참석자/안건
- 주요 논의
- 결정사항
- 액션아이템
- 책임자/기한
- 미해결 이슈
