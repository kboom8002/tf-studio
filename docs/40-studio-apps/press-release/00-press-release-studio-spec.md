# Press Release Studio Spec

## 핵심 목적
공식 보도자료 초안을 빠르고 일관되게 생성하고, 과장·허위·비방·구조 누락을 통제한다.

## 핵심 사용자
홍보팀, 마케팅 커뮤니케이션, 대외협력, 스타트업 대표/PMM, PR 에이전시

## TF8 매핑
- T: 공식 보도자료 작성
- A: 산업 기자 + 일반 대중
- S: 발표 맥락, 시급성, 배포 목적
- K: 수치, 사실, 배경, 코멘트 원문
- F: 헤드라인 → 리드문 → 본문 → 배경/맥락 → 임원 인용 → 마무리
- L: 건조하고 객관적인 공식 언론보도체
- O: 스트레이트 기사형
- W: 과장 금지, 비방 금지, 근거 없는 단정 금지

## 출력 계약
- 헤드라인
- 리드문
- 본문 2개 이상
- 임원 코멘트(선택)
- 마무리/문의(선택)

## Critical Blocker
- 핵심 수치 근거 전무
- 타사 비방
- 검증되지 않은 약속/효과 보장
- 발표 주체/행위/대상/의미가 불명확
