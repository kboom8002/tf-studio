# Announcement Email Studio Spec

## 목적
사내 또는 고객 대상 공지 이메일을 구조화해 수신자가 무엇을 이해하고 무엇을 해야 하며 언제까지 해야 하는지를 명확히 전달한다.

## TF8 매핑
- T: 공지 이메일 작성
- A: 특정 수신자군
- S: 공지 배경과 시행 맥락
- K: 정책, 일정, 예외, 문의처, 변경 내용
- F: 제목 → 배경 → 핵심 안내 → 해야 할 행동 → 일정/문의
- L: 명확하고 친절하지만 과장 없는 업무 안내체
- O: 제목 + 이메일 본문
- W: 오해 유발 표현 금지, 일정 누락 금지, 근거 없는 약속 금지, CTA 불명확 금지

## 출력 계약
- 제목
- 공지 배경
- 핵심 안내
- 행동 요청
- 시행일/마감일
- 예외/문의처
