이 스킬의 목적은 작업 시작 전에 문서 참조 경로를 빠르게 잠그는 것이다.

절차:
1. 작업을 Core OS / Studio App / Governance 중 어디에 속하는지 분류한다.
2. source of truth 문서를 3~7개 고른다.
3. 불변조건을 5개 이내로 뽑는다.
4. 구현 전에 확인할 질문을 정리한다.

출력 형식:
- 작업 분류
- 먼저 읽을 문서
- 핵심 불변조건
- 구현 전 확인 질문
